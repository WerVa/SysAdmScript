1. First way

a) Install pipreqs

```python
pip install pipreqs # Python2
```

```python
pip3 install pipreqs # Python3
```

b) Usage pipreqs

```python
pipreqs /path/to/project # Python2
```

```python
python3 -m  pipreqs /path/to/project # Python3
```

2. Second way

```
pip3 freeze > requirements.txt  # Python3
pip freeze > requirements.txt  # Python2
```
